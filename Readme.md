## 💻 About the Project

This project was developed as part of a training program focused on sharpening Front-end fundamentals. The goal was to create a visually appealing, semantically structured interface styled with classic CSS techniques, simulating the layout of a premium coffee shop.

## 🚀 Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Styling, box-model manipulation, and absolute/relative positioning.

## 🧠 Key Learnings & Overcome Challenges

During development, I solidified important Front-end concepts:

- **Pure CSS Parallax Effect:** Implementing a fixed background using only CSS (`background-attachment: fixed`) to create depth.
- **Advanced Positioning:** Mastering the document flow using `transform: translateY` for mathematically perfect centering.
- **Specificity and Clean Code:** Organizing the CSS hierarchy to avoid style conflicts and maintain scalable code, understanding the cascade without relying on visual cheats.

## 🛠️ How to run the project

Since this is a purely static project (HTML/CSS), no complex installation is required.

1. Clone this repository.
2. Open the project folder.
3. Double-click the `index.html` file to open it in your default web browser.
