## 💻 Sobre o Projeto

Este projeto foi desenvolvido como parte de um treinamento focado em aprimorar os fundamentos do Front-end. O objetivo foi criar uma interface visualmente agradável, estruturada semanticamente e estilizada com técnicas clássicas de CSS, simulando o layout de uma cafeteria de alto padrão.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização, manipulação de box-model e posicionamento absoluto/relativo.

## 🧠 Principais Aprendizados e Desafios Superados

Durante o desenvolvimento, consolidei conceitos importantes do Front-end:

- **Efeito Parallax Puro:** Implementação de fundo fixo usando apenas CSS (`background-attachment: fixed`) para criar profundidade.
- **Posicionamento Avançado:** Domínio sobre o fluxo do documento com `transform: translateY` para centralizações matemáticas perfeitas.
- **Especificidade e Clean Code:** Organização da hierarquia do CSS para evitar conflitos de estilo e manter o código escalável, compreendendo a cascata sem depender de trapaças visuais.

## 🛠️ Como rodar o projeto

Como é um projeto puramente estático (HTML/CSS), não é necessária nenhuma instalação complexa.

1. Clone este repositório.
2. Abra a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador padrão.
